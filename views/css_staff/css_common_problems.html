Positioning in css

position:inline

The default value for elements. Think of elements like span, em, or I am b or a tag I am tag and how wrapping text in those elements within a string of text doesn't break the flow of the text.

An inline element will accept margin and padding, but the element still sits inline as you might expect. Margin and padding will only push other elements horizontally away, not vertically.
